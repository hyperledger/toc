Created by Attila Klenik, last modified by Gari Singh on Oct 16, 2021

Two maintainers and one significant contributor working on the project.

integrations.

# Overall Activity in the Past Quarter

Identifying critical design/quality issues to improve: robustness,
debugging experience, testability, integrations, and user experience.

# Current Plans

-   Compliance with common repo requirements
-   Addressing the above-mentioned issues (current code quality still
makes first contributions a challenge)
-   Improve user (workload implementation) experience, done in the
context of a mentorship project
